graphics:: graphic package

Graphics is a great strength of R. The graphics package is part of the standard distribution and contains many useful functions for creating a variety of graphic displays. This chapter focuses on those functions, although it occasionally suggests other packages. In this chapter's See Also sections I mention functions in other packages that do the same job in a different way. I suggest that you explore those alternatives if you are dissatisfied with the basic function.

Graphics is a vast subject, and I can only scratch the surface here. If you want to delve deeper, I recommend R Graphics by Paul Murrell (Chapman & Hall, 2006). That book discusses the paradigms behind R graphics, explains how to use the graphics functions, and contains numerous examples.

You can see the documentation of graphics package in the following link.

The useful functions in the graphics package is listed below:

  • plot() : Generic plotting function
  • boxplot() : Create a box plot
  • hist() : Create a histogram
  • qqnorm() : Create a quantile-quantile (Q-Q) plot
  • curve() : Graph a function
  • points() : Add points
  • lines() : Add lines
  • abline() : Add a straight line
  • text() : Add text

plot(), boxplot(), hist(), qqnorm(), and curve() generate the new plots, but other functions did not create a new plot, instead they add layers in the previous plot.

In addtion to the base of R pachage, there are other packages for the graphic purpose. zoo package is customized for displaying a time series, lattice package can be an alternative to traditional graphics. ggplot2 package provides yet another graphics paradigm, which is called the Grammar of Graphics.
